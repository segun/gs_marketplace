# A simple NFT Market Place.

    Built with
        node v14.17.0
        hardhat v2.5.0
        solidity 0.8.3
        Mac OSX 11.1

    Tested on
        node v10.13.0
        Mac OSX 10.14

## Building & Running Test Cases

    run npm install to install all dependencies
    run npx hardhat compile to compile all solidity files
    run npx hardhat test to run all test cases.
